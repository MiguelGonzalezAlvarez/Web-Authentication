Para hacer funcionar el servidor es necesario tener node_modules instalado lo cual se puede hacer haciendo
npm install en la carpeta server 

Para lanzar el servidor usamos los comandos:
    - npm run dev
    - npm start

Si queremos usar el despliegue de heroku tenemos que sacar a parte la carpeta server y teniendo en public lo que
nos haya generado el ng build en la carpeta dist
